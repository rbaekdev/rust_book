## Installation
 
 Mac

```js
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
// optional
$ xcode-select --install
```

 Windows

[Install Link](https://www.rust-lang.org/tools/install)

 ```js title="Update"
$ rustup update
```
